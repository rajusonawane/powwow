## Purpose of application:

 **Purpose of this sample application is to demonstrate handling of JS alert and confirm popup** 
  ### Run web server
 - Go to SampleSourceApplications\PopupWeb folder
 - run `npm install` on console if node modules are not installed
 - Start node service `node server.js`
 
 ### Running the sample app
 - Go to server folder and run `npm start`
 - Open the application in  smart UX studio.
 - Run server on 3000 port.
 - Click on preview and put server URL as ws://localhost:3000.

  ### How to handle JS popup
  1. Handling Alert
   ```
   page
        .action("alert", "alert")
        .onTimer(500)
        .onAlert()
        .extract("alert")
        .screen('popup')
 ```
 
 2. Handle confirm popup
    - Pre-select OK or Cancel button

        * Pre-select Ok button
    ```
            page
                .confirmClickOK()
                .action("alert", "confirm")
                .data(function (data) {
                    data.confirmPopupText = "You pressed OK!"
                })
                .screen('popup');
     ```
                *  Pre-select Cancel button

    ```
                    page
                        .confirmClickCancel()
                        .action("alert", "confirm")
                        .data(function (data) {
                            data.confirmPopupText = "You pressed OK!"
                        })
                        .screen('popup');
     ```

    -   Handle the event as popup showed and send the response to UI

        * Handler for event that says it showed, and send the message to the UI, click on cancel button on confirm popup.
        ```
            page.onConfirm("_confirm")
                .onTimer(600)
                .data(function (data) {
                    data.confirmMessage = { type: "Confirm", message: data._confirm, action: params.action };
                    delete data._confirm;
                })
                .extract("alert")
                .screen('popup');
        ```
        * Handler for event that says it showed, and send the message to the UI, click on OK button on confirm popup.
            ```
                page.onConfirmOK("_confirm")
                    .onTimer(600)
                    .data(function (data) {
                        data.confirmMessage = { type: "Confirm", message: data._confirm, action: params.action };
                        delete data._confirm;
                    })
                    .extract("alert")
                    .screen('popup');
            ```
